## [1.0.1] - 2026-07-24

### Added
- Introduced DiceBear-powered profile avatars for every user.
- Added avatar customization through an interactive profile customization modal.
- Added support for multiple avatar styles (e.g., geometric, human, animal, abstract).
- Added one-click avatar regeneration with instant preview.
- Persisted avatar style and generated seed locally, allowing personalized avatars without requiring backend or database changes.

### Changed
- Redesigned the profile avatar experience to eliminate the need for uploaded profile pictures.
- Improved profile personalization while maintaining a clean and lightweight implementation.
- Enhanced avatar interactions with hover effects and smoother UI transitions.

### Fixed
- Minor UI refinements and usability improvements.

## [1.0.0] - 2026-07-24

### 🎉 Initial Release

This marks the first stable release of **LedgerLeaf**, a modern personal finance management application focused on budgeting, expense tracking, savings management, and financial reporting.

### Added

#### Core Features
- User authentication and account management.
- Financial dashboard with account overview.
- Budget management.
- Expense tracking.
- Savings management.
- Financial reporting and analytics.

#### Data Management
- Export reports in PDF, Excel, and CSV formats.

#### User Experience
- Responsive interface for desktop and mobile devices.
- Modern dashboard and navigation.
- Landing page for introducing the application.
- Comprehensive Help Center and project documentation.

#### Deployment
- Production-ready deployment configuration for Railway.

### Fixed
- Updated deployment configuration (`composer.json`) to ensure compatibility with Railway.
